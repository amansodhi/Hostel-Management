# Hostel-Management
Hostel Management System is a customize and 
user-friendly software for hostel which provides
hostel information,hostel room information, and 
hostel accounts information .To generate reports 
on hostel occupancy and financial reports We have
two modules  Admin and Student Admin: can add page ,
manage fee ,set the fes Student: can seee the room 
info, see the mess and hostel bill, can register the room  
To provide quick and efficient means for gathering the student
information along with their rooms,fees of student. 
This project offers users to enter the data through simple 
and interactive manner  User is provided the option of only 
view the records he/she entered earlier   Data Storage and 
retrieval will become faster and easierr to maintain
